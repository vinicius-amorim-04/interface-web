HTML

O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para estruturar e organizar o conteúdo de páginas web. Diferente de linguagens de programação, o HTML não cria algoritmos ou amanipula dados, mas define a estrutura de elementos como textos, imagens, links, tabelas e formulários que compõem uma página.

A história do HTML:

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

Como funciona:

O HTML foi a primeira linguagem de nível mundial, porem não é a única. Existem muitas outras linguagens destinadas á criação de paginas da web, porém o HTML ainda prevalece. Atualmente já é possível integrar varias linguagens na mesma pagina da Web, sendo possível usar duas ou mais linguagens no mesmo site.

Para criar e editar códigos em HTML é necessário qualquer editor de texto comum, como bloco de notas. Para testar os códigos, basta salvar o arquivo em formato .HTML e executar. Para o teste é necessário ter um navegador configurado como padrão. Não é necessária internet, pois o arquivo com os códigos esta na maquina onde esta sendo executado.

Em quais áreas da programação o HTML é usado?

Interface de Usuário em Apps Script: O serviço HTML do Google Apps Script permite criar interfaces personalizadas para Documentos, Planilhas e Formulários Google.

Desenvolvimento Web Frontend: Esta é a principal aplicação. O HTML é usado para criar a estrutura, organizar conteúdo (texto, imagens, vídeos, tabelas) e definir elementos semânticos em sites

Documentos e Aplicações Desktop: HTML pode ser usado na criação de documentos estruturados (HTML5) e, com tecnologias como Electron, para criar aplicações desktop.

Aplicativos Híbridos (Mobile): Ferramentas como Ionic ou React Native utilizam HTML, CSS e JavaScript para desenvolver aplicativos móveis para Android e iOS, rodando o código dentro de um "container" nativo.

Aplicações Web e Web Apps: Utilizado para criar a interface de usuário (UI) de aplicações que rodam no navegador. É frequentemente emparelhado com CSS (estilização) e JavaScript (interatividade).

História da internet

Quando surgiu a internet?

O surgimento da internet está ligado à disputa tecnológica entre Estados Unidos e União Soviética durante a Guerra Fria. Após o lançamento do satélite Sputnik pelos soviéticos, o governo norte-americano decidiu investir em projetos tecnológicos para fortalecer sua defesa. Em 1958 foi criada a Advanced Research Projects Agency (Arpa), que desenvolveu um sistema para enviar informações entre centros militares, pesquisadores e o Pentágono, visando também a proteção do país em caso de guerra nuclear.

Esse projeto recebeu o nome de Arpanet e foi conduzido por Robert Taylor e Lawrence Roberts, com a colaboração de vários cientistas, utilizando a tecnologia de redes WAN para transmissão de dados. A Arpanet foi inaugurada em 1969 com apenas quatro computadores conectados. A primeira mensagem foi enviada da Universidade da Califórnia (UCLA) para o Instituto de Pesquisa Stanford (SRI), mas o teste inicial falhou quando o sistema travou ao receber a letra “g”. Pouco depois, um segundo teste foi realizado com sucesso, estabelecendo a primeira conexão permanente entre as duas instituições.
Evolução da internet:
Na década de 1970, a Arpanet era usada por pesquisadores e militares, e o e-mail tornou-se o principal meio de comunicação. Inicialmente, a rede utilizava o protocolo NCP, mas com o aumento do uso foi criado o protocolo TCP/IP, implantado em 1983, que permitiu padronizar e melhorar a transmissão de dados.

Com o crescimento da rede, os militares criaram uma rede separada chamada Milnet por motivos de segurança. A Arpanet continuou sendo usada por pesquisadores e passou a ter uso comercial na década de 1980. Em 1986, foi criada a NSFNET, que ampliou o acesso e se tornou a base da internet moderna.

A Internet
A internet também transformou os meios de comunicação tradicionais, como telefone, música, cinema e televisão, que passaram a funcionar ou ser distribuídos pela rede. Surgiram novos serviços, como chamadas pela internet e televisão online, enquanto jornais, livros e outras publicações passaram a utilizar sites, blogs e plataformas digitais. Além disso, a internet possibilitou novas formas de interação entre as pessoas, como mensagens instantâneas, fóruns e redes sociais, e impulsionou o crescimento do comércio eletrônico.

Apesar de ser uma rede mundial, a internet não possui um controle central único. Cada rede define suas próprias regras de uso, enquanto algumas organizações ajudam a manter seu funcionamento técnico. A gestão dos nomes de domínio e endereços é feita pela Internet Corporation for Assigned Names and Numbers (ICANN), e o desenvolvimento e padronização dos principais protocolos da internet são realizados pela Internet Engineering Task Force (IETF), que reúne especialistas de vários países.
